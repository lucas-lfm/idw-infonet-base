Listas não ordenadas

Listas não ordenadas são definidas com a tag <ul>, onde a ordem dos itens não importa. Para definir cada item, usamos a tag <li>. ✨

Principais tecnologias para front-end

  • HTML
  • CSS
  • JavaScript
  • Bootstrap
  • React

Listas ordenadas

Listas ordenadas são definidas pela tag <ol>, onde a ordem importa. Cada item também é definido pela tag <li>.

Minhas linguagens preferidas

  1. Java
  2. JavaScript
  3. Python
